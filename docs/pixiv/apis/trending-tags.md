## 简要描述

- 热门Tag接口

## 请求URL
- ` /v1/trending-tags/illust?filter=for_android `
  
## 请求方式
- GET 

## 参数

|参数名|可选值|类型|说明|
|:- |:- |:- |:- |
|filter|`for_android`|string||
